# 📊 Customer Churn Analysis Dashboard (Power BI)

## 📌 Project Overview
This project focuses on analyzing customer churn behavior in a telecommunications company using Power BI. The objective is to identify key factors that contribute to customer churn and provide actionable insights to improve customer retention.

---

## 🎯 Objectives
- Analyze customer data to understand churn patterns  
- Identify high-risk customer segments  
- Build an interactive dashboard for business insights  
- Provide data-driven recommendations to reduce churn  

---

## 🛠️ Tools & Technologies
- Power BI (Data Visualization & Dashboard)
- Power Query (Data Cleaning & Transformation)
- DAX (Data Analysis Expressions)

---

## 📂 Dataset
- Telco Customer Churn Dataset  
- Contains information about:
  - Customer demographics  
  - Account details  
  - Services used  
  - Contract and payment methods  
  - Churn status  

---

## 🔧 Data Preparation
- Removed null and inconsistent values  
- Converted data types (e.g., tenure, total charges)  
- Created new columns:
  - Tenure Group (New, Mid-term, Long-term)
  - Churn Flag (1 = Yes, 0 = No)  

---

## 📊 Dashboard Features

### 1. Churn Rate Overview
- Total Customers  
- Churned Customers  
- Churn Percentage  

### 2. Customer Demographics
- Gender distribution  
- Partner status  
- Dependent status  

### 3. Customer Tenure Analysis
- Distribution of customers based on tenure  
- Identification of high churn in early stages  

### 4. Churn Analysis
- Churn by Contract Type  
- Churn by Payment Method  
- Churn by Internet Service  

---

## 📈 Key Insights
- Customers with **month-to-month contracts** have the highest churn rate  
- Customers using **electronic check payments** show higher churn behavior  
- **New customers** are more likely to churn compared to long-term customers  
- Certain service types contribute more to customer churn  

---

## 💡 Conclusion
This project demonstrates how data visualization and analysis can help businesses understand customer behavior and take proactive steps to reduce churn.
